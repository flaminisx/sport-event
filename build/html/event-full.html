Загрузить положения

Для всіх гостей - вітання від учасників Олімпійської збірної України, святковий концерт гурту "ТІК", дитячі анімаційні зони, герої мультфільмів і благодійна лотерея на найбільшій розважально-пішохідній зоні Лівобережжя - Русанівській набережній.

Для учасників - найбільший призовий грошовий фонд за історію проведення "Забігу у вишиванках", спортивна футболка-вишиванка та сувенірна медаль, номер з чиповой системою - в стартовому пакеті.
